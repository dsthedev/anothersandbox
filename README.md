## Contribution

```bash
node: "=18.x"
yarn: ">=1.15"
```

```bash
yarn redwood dev` # Run devtools and open [local site](http://localhost:8910/)
yarn rw upgrade` # Upgrade core framework

yarn workspace web add marked
yarn workspace api add better-fs

yarn rw g page page-title
```
